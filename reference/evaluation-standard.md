## 一、核心结构框架

### 1.1 标准章节（7大核心章节）

| 序号 | 章节名称 | 权重 | 核心内容 | 质量要求 |
|------|----------|------|----------|----------|
| 1 | 执行摘要 | 10% | 交易概述与背景、关键发现与风险提示、投资建议与结论、核心价值点总结 | 简明扼要，突出重点，为决策层提供快速概览 |
| 2 | 公司基本情况 | 15% | 公司简介、历史沿革、股权结构、组织架构、人力资源情况 | 信息准确完整，历史沿革清晰，股权关系明确 |
| 3 | 行业与竞争 | 15% | 行业概况、产业政策与法规、行业发展现状、竞争格局分析、行业壁垒、市场容量预测 | 数据来源可靠，分析客观深入，竞争格局清晰 |
| 4 | 业务与技术 | 20% | 主营业务介绍、核心产品与服务、技术与研发、知识产权、业务模式、客户与供应商分析 | 技术路线清晰，研发能力评估准确，知识产权布局完整 |
| 5 | 财务情况 | 20% | 财务报表、关键财务指标分析、盈利能力评估、财务预测、财务风险识别 | 数据准确，分析深入，预测合理，风险识别到位 |
| 6 | 投资价值分析 | 10% | 投资亮点、估值方法与测算、投资回报预测、投资方案建议 | 估值方法合理，假设清晰，测算过程可验证 |
| 7 | 风险与对策 | 10% | 行业风险、公司特有风险、财务风险、法律风险、风险缓释措施 | 风险识别全面，分析客观，对策可行 |

### 1.2 可选章节

| 章节名称 | 适用场景 |
|----------|----------|
| 结论与建议 | 需要给出明确投资建议的报告 |
| 其他事项与说明 | 需要补充说明特殊情况或限制条件 |
| 投后管理 | 股权投资项目 |
| ESG评估 | 2026年已成为标配 |

## 二、关键要素清单

### 2.1 财务尽调要素

| 要素 | 说明 | 重要性 |
|------|------|--------|
| 资本化分析 | 股权结构、市值、流通股比例、机构持股 | 高 |
| 收入利润趋势 | 过去3-5年营收、净利润、毛利率、净利率趋势 | 高 |
| 估值倍数 | P/E、PEG、P/B、EV/EBITDA、P/S等估值指标 | 高 |
| 资产负债表 | 现金水平、偿债能力、债务结构、债务权益比 | 高 |
| 现金流分析 | 经营性现金流、投资性现金流、筹资性现金流 | 高 |
| 财务预测 | 未来3-5年收入、利润预测，关键假设说明 | 中 |

### 2.2 业务尽调要素

| 要素 | 说明 | 重要性 |
|------|------|--------|
| 竞争格局 | 主要竞争对手、市场份额、差异化优势 | 高 |
| 管理层评估 | 创始人背景、管理团队履历、持股比例、稳定性 | 高 |

| 供应商分析 | 主要供应商、采购额占比、供应商集中度、替代性 | 中 |
| 商业模式 | 盈利模式、收入来源、成本结构 | 高 |

### 2.3 技术尽调要素（半导体行业）

| 要素 | 说明 | 重要性 |
|------|------|--------|
| 核心技术 | 技术路线、技术壁垒、技术先进性 | 高 |
| 研发团队 | 研发人员数量、学历结构、核心技术人员背景 | 高 |
| 知识产权 | 专利数量、专利质量、专利布局、侵权风险 | 高 |
| 研发投入 | 研发费用占营收比例、研发管线 | 高 |

### 2.4 法律尽调要素

| 要素 | 说明 | 重要性 |
|------|------|--------|

| 重大合同 | 销售合同、采购合同、借款合同、担保合同 | 高 |

| 诉讼与仲裁 | 历史诉讼、潜在诉讼风险、执行情况 | 高 |
| 合规性 | 行业资质、环保合规、安全生产、劳动用工 | 高 |

### 2.5 风险要素

| 要素 | 说明 | 重要性 |
|------|------|--------|
| 行业风险 | 监管变化、技术替代、市场萎缩、竞争加剧 | 高 |
| 公司风险 | 客户集中、供应商依赖、管理层变动、技术迭代 | 高 |
| 财务风险 | 现金流紧张、债务违约、盈利能力下滑 | 高 |
| 法律风险 | 诉讼纠纷、合规问题、知识产权纠纷 | 中 |

## 三、行业特殊要求

### 3.1 半导体行业

**重点关注**：
- 技术壁垒与创新能力
- 专利布局与侵权风险
- 研发团队稳定性与实力
- 供应链安全（特别是设备、材料国产化）
- 客户认证周期与黏性
- 地缘政治风险
- 产能爬坡能力
- 良率水平

**财务特征**：
- 高研发投入（通常占营收10-20%）
- 长投资回报周期
- 存货跌价风险
- 资本密集型特征
- 客户认证周期长

### 3.2 科技互联网行业

**重点关注**：
- 用户指标（DAU/MAU/ARPU）
- 商业模式与变现能力
- 数据安全与隐私合规
- 知识产权侵权风险
- 反垄断风险
- 平台生态与网络效应

### 3.3 制造业

**重点关注**：
- 产能利用率与瓶颈
- 资产质量与成新率
- 供应链效率
- 环保合规
- 安全生产
- 库存管理

### 3.4 金融服务业

**重点关注**：
- 监管牌照完整性
- 风险管理能力
- 资产质量
- 资本充足率
- 合规体系

### 3.5 消费品行业

**重点关注**：
- 品牌价值与美誉度
- 渠道管理与控制力
- 产品创新能力
- 舆情风险
- 季节性波动

## 四、质量评估标准

### 4.1 评估维度

| 维度 | 权重 | 评分要点 |
|------|------|----------|
| 完整性 | 20% | 信息覆盖全面，无重大遗漏；各模块齐全，结构完整；历史数据时间跨度充足（至少3年）；信息来源充分可靠 |
| 准确性 | 25% | 财务数据准确，来源可靠；关键假设合理，有依据；计算过程可验证；数据交叉验证一致 |
| 深度性 | 20% | 分析深入，有洞察，非简单罗列；识别关键风险与价值驱动因素；提出建设性建议；行业对比分析充分 |
| 实用性 | 20% | 对投资决策有实质帮助；风险揭示全面清晰；建议具体可行；估值方法合理 |
| 可读性 | 15% | 结构清晰，逻辑严谨；表述专业，语言简洁；图表运用恰当；重点突出，易于快速阅读 |

### 4.2 评分标准

| 维度 | 优秀 | 良好 | 合格 | 不合格 |
|------|------|------|------|--------|
| 完整性 | 所有核心模块完整，信息覆盖度>90% | 核心模块齐全，信息覆盖度70%-90% | 基本模块齐全，信息覆盖度60%-70% | 缺失重要模块，信息覆盖度<60% |
| 准确性 | 数据准确率>95%，所有数据有来源标注 | 数据准确率90%-95%，大部分数据有来源 | 数据准确率80%-90%，部分数据有来源 | 数据准确率<80%，缺少来源标注 |
| 深度性 | 深度分析，有独特洞察，建议可执行 | 分析较深入，有一定洞察 | 基本分析到位，洞察一般 | 仅罗列信息，缺乏分析 |
| 实用性 | 直接支持决策，风险建议可操作 | 对决策有较好帮助 | 对决策有一定参考价值 | 实用性差，难以指导决策 |
| 可读性 | 结构清晰，表述专业，重点突出 | 结构较好，表述清晰 | 基本结构清晰，表述通顺 | 结构混乱，表述不清 |

### 4.3 总体评分标准

| 评级 | 条件 |
|------|------|
| 优秀 | 总分≥90分，且各维度均≥良好 |
| 良好 | 总分80-89分，且无不合格维度 |
| 合格 | 总分60-79分，且无严重不合格维度 |
| 不合格 | 总分<60分，或有严重不合格维度 |

## 五、撰写规范

### 5.1 基本原则

- **客观独立**：保持客观立场，避免利益冲突
- **充分披露**：完整披露信息来源、假设条件、局限性
- **风险导向**：重点关注风险揭示和风险缓释
- **决策导向**：为投资决策提供实质性帮助
- **专业规范**：使用专业术语，表述准确严谨

### 5.2 数据来源要求

| 数据类型 | 优先来源 |
|----------|----------|
| 财务数据 | 经审计财务报表 |
| 市场数据 | 注明数据来源和时间 |
| 行业数据 | 权威机构研究报告 |
| 访谈信息 | 记录访谈对象、时间、内容 |

### 5.3 报告要素

- **标题**：明确投资标的和时间范围
- **目录**：结构清晰，页码准确
- **正文**：逻辑严谨，数据准确
- **附件**：重要文件、数据表格
- **免责声明**：说明报告局限性和免责事项

## 六、常见问题与改进建议

| 问题 | 改进建议 |
|------|----------|
| 信息罗列多，分析少 | 增加深度分析，提炼洞察，提出判断和建议 |
| 风险揭示不足或避重就轻 | 客观全面揭示风险，提供风险缓释措施 |
| 财务数据缺少来源标注 | 注明所有数据来源，提高可信度 |
| 估值方法单一或假设不合理 | 采用多种估值方法交叉验证，明确假设依据 |
| 行业分析泛泛而谈 | 深入分析细分市场，提供具体数据和案例 |
| 缺少竞争对手对比 | 明确主要竞争对手，进行多维度对比分析 |
| 核心技术介绍不清晰 | 用通俗语言解释技术原理，强调技术壁垒和先进性 |
| 缺少投资建议或建议不明确 | 给出明确的投资建议，说明理由和风险提示 |

## 八、参考案例亮点

### 8.1 德智新材报告

**优点**：
- 历史沿革清晰：每次股权变更的时间、价格、比例详细记录
- 行业分析深入：详细分析碳化硅材料发展状况、市场容量测算
- 技术介绍专业：化学气相沉积法的技术原理、工艺流程、性能指标
- 业务拓展明确：列出客户名称、订单金额、测试进展
- 风险识别到位：行业风险、技术风险、市场风险分析

### 8.2 北一半导体报告

**优点**：
- 信息详实：母公司和子公司的历史沿革、股权结构非常详细
- 产品矩阵完整：详细介绍产品系列、技术参数、应用领域
- 财务数据全面：包含完整财务报表、财务分析、盈利预测
- 竞争分析充分：与英飞凌等龙头的性能对比
- 风险识别具体：列出主要问题和风险

### 8.3 广东致能报告

**优点**：
- 概要清晰：开篇概要章节提炼核心价值
- 技术路线明确：详细介绍GaN垂直结构器件的技术优势
- 团队背景扎实：详细列出核心团队的教育和工作经历
- 行业发展前瞻：分析GaN应用领域和市场空间
- 融资历程清晰：详细记录每轮融资的时间、金额、估值
